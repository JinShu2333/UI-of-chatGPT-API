### Requirements

* openai>=0.27.0

### How to use

* Install package
  ```
  pip install openai
  ```
* Replace with your secret key
* Run the script
  ```python
  python chatUI.pyw
  ```

### API key

* Get key from https://platform.openai.com/
* Click the avatar on the right upper conner, View API keys


### Customization

* Modify PROMPT to determine AI's character.
